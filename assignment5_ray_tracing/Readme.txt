1. open main.cpp on codeblocks
2. then double click on OpenGL.cbp to link the libraries to main .cpp
3. run the code
4. press keyboard button 1 to 5 to interact with the camera and postion.
5. press keyboard button 0 to generate the output (wait for a few seconds to see the output file "out.bmp").
6. open and see the out.bmp file.
